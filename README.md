# Amazon Clone Website

This is an Amazon-inspired e-commerce website built using **HTML, CSS, and JavaScript**.  
The project was created to practice frontend development and understand how real e-commerce websites structure their user interface.

## Features

- Homepage with product sections
- Image slider using JavaScript
- Product detail page
- Shopping cart page
- Sign in page
- Sign up page
- Multi-page navigation
- Amazon-style UI layout

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Project Pages

- Home Page
- Product Page
- Cart Page
- Sign In Page
- Sign Up Page
